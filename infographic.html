Every Production Bot Needs
non-negotiables learned the hard way
✓
Position isolation — bots sharing one HL account must only close their own trades (read peer state files)
✓
Circuit breaker — pause on daily loss > 4%, resume after cooldown window
✓
Round-trip fees — always subtract 0.03% (maker × 2) from PnL, not just one side
✓
Graceful SIGTERM — save state, never close positions on shutdown
✓
Blacklist thin markets — no BANANA, SOPH, anything < $500k OI
✓
Per-coin cooldowns — 3h minimum after closing to avoid re-entry churn
✓
State persistence — JSON every cycle, SQLite per trade (survive restarts)
✓
429 retry w/ backoff — Hyperliquid rate limits will bite the first week
